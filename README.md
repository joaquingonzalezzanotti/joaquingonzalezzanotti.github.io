# JGZ Portfolio

Sitio personal y portfolio de **Joaquin Gonzalez Zanotti**.

## Contenido

- `index.html`: landing principal (perfil, portfolio, stack, contacto).
- `Joaquin Gonzalez Zanotti CV.pdf`: CV actualizado con enlaces clickeables a LinkedIn y Portfolio.
- `sitemap.xml`: mapa del sitio para indexacion.

## Como usar

1. Abrir `index.html` en navegador para ver el sitio localmente.
2. Publicar estos archivos en tu hosting estatico (GitHub Pages, Vercel, Netlify, etc.).

## Enlaces

- LinkedIn: `https://www.linkedin.com/in/joaquin-gonzalez-zanotti/`
